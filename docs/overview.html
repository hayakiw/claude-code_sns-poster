ノードの種別

🟪 AIエージェント／ Scout・Writer・Thumbnail・Analyst（自動）
🟩 外部キュー／ Typefully（自動登録・人が公開）
⬜ 判定・状態／ /goal 達成判定・公開済み
🟧 人間承認／ 公開承認（公開のみ）

しくみ

  • /loop が量を出す：Scout→Writer→Thumbnail→Typefully登録を日次で自動巡回
  • /goal が質と方向を制御：fit ≥ 0.6 の上位 N 件だけ承認へ
  • 統括・採点・KPIは sns-cycle スキル（コード無し）が担当
  • 状態は state/ の 2 ファイル（goal.json / playbook.md）
🔒 安全性：Typefully には「下書き」を作るだけ。
X への公開は 人間が Typefully 上で実行。自動公開は一切しない（無応答は HOLD）。
